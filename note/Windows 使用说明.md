Windows 8 使用技巧
------------------

### PowerShell

直接输入 powershell 即可进入 powershell。

### tasklist -- 当前进程列表

    tasklist
    tasklist /m

### Windows 8 中有没有 wc 命令?

进入 PowerShell，然后试试下面命令：

    # wc -l -w -c
    measure -l -w -c
    # wc -l
    (cat myfile.txt).count
